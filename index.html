Ưu tiên sửa lỗi hiện tại

Nguyên nhân trang cũ chỉ hiện tiêu đề là GitHub Pages đang phục vụ nội dung root/README thay vì giao diện web đã thiết kế trong app Next.js.

  1. Critical: thêm `index.html` production-ready tại root để URL GitHub Pages hiển thị landing page ngay.
  2. High: giữ app Next.js trong `apps/web` để tiếp tục phát triển dashboard/auth/runtime mà không phá deploy hiện tại.
  3. Medium: bổ sung workflow Pages chính thức khi app loại bỏ API động hoặc có backend tách riêng.
  4. Low: tinh chỉnh motion, illustration, logo và docs content theo brand guideline.
